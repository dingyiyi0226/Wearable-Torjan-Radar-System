# Wearable Trojan Radar System

2020 IEEE AP-S Student Design Contest - Team Troy


## How to use

1. Build up the system with the [instruction](Instruction.pdf)

2. Edit the pin setting in `config.py`

3. Run main process

    ```
    python3 main_control.py
    ```
